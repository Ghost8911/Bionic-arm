# Bionic-arm
Im trying to build an arm
The goal of this project is to design and assemble a functional 4-DOF (Degrees of Freedom) bionic arm using an ESP32 microcontroller and SG90 servo motors, transitioning from a cardboard and wood prototype to a more durable mechanical structure. By utilizing a dual-system control architecture, the arm interprets real-time commands sent from a Python-based keyboard interface on a PC via serial communication to execute precise movements in the shoulder, elbow, wrist, and a custom-linked two-finger gripper. This journey focuses on mastering the intersection of hardware integration, such as managing power requirements and signal stability on a breadboard, and software refinement, including debugging serial data streams and mapping intuitive user controls to physical robotic motion.
